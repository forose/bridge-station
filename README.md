# 简介
桥头堡（bridge—station）是一个基于netty实现的、开源的跨局域网通信平台，实现将局域网内的服务通过内网穿透、公网端口映射等方式搬移到个人电脑、另一个局域网，实现跨局域网合作、创建局域网之间私密连接的虚拟网络工具。
